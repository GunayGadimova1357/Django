# Функции как объекты первого класса (First-Class Functions) в Python

В Python **функции являются объектами первого класса (first-class objects)**.
Это означает, что функции **ведут себя так же, как и любые другие переменные** (например, числа, строки или списки).

Другими словами, функции можно:

* присваивать переменным
* хранить в структурах данных
* передавать как аргументы
* возвращать из других функций
* создавать во время выполнения программы

Это делает Python **гибким и мощным языком**, позволяющим использовать функциональные подходы к программированию.

---

# Основные возможности функций как объектов первого класса

## 1. Присваивание функции переменной

Функцию можно сохранить в переменной и вызывать через эту переменную.

```python id="p20u0i"
def greet():
    print("Hello")
```

```python id="5t9cnk"
say_hello = greet
say_hello()
```

### Результат

```
Hello
```

Здесь переменная `say_hello` хранит **ссылку на функцию** `greet`.

---

# 2. Хранение функций в структурах данных

Функции можно хранить в:

* списках
* кортежах
* словарях

### Пример

```python id="o5ajfp"
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

operations = [add, multiply]

print(operations[0](2, 3))  # 5
print(operations[1](2, 3))  # 6
```

---

# 3. Передача функций как аргументов (Функции высшего порядка)

Функция может принимать **другую функцию в качестве аргумента**.

Такие функции называются **функциями высшего порядка (Higher-Order Functions)**.

### Пример

```python id="o3imab"
def apply_function(func, value):
    return func(value)
```

```python id="u2j5hj"
def square(x):
    return x * x

print(apply_function(square, 5))
```

### Результат

```
25
```

---

# Примеры встроенных функций высшего порядка

Python содержит несколько встроенных функций, которые принимают функции как аргументы:

* `map()`
* `filter()`
* `sorted()`

### Пример

```python id="0yufih"
numbers = [1, 2, 3, 4]

squared = list(map(lambda x: x**2, numbers))

print(squared)
```

---

# 4. Возврат функции из другой функции

Функция может **создавать и возвращать другую функцию**.

Это используется в **замыканиях (closures)**.

### Пример

```python id="k87pme"
def outer_function(text):

    def inner_function():
        print(text)

    return inner_function
```

```python id="0qccfc"
my_func = outer_function("Hello")

my_func()
```

### Результат

```
Hello
```

Внутренняя функция **запоминает переменную `text`**, даже после завершения внешней функции.

---

# 5. Создание функций во время выполнения (runtime)

Функции могут быть **созданы динамически во время работы программы**.

### Пример с `lambda`

```python id="8acqvt"
multiply = lambda a, b: a * b

print(multiply(3, 4))
```

### Результат

```
12
```

---

# Почему это важно

Благодаря тому, что функции являются объектами первого класса, Python поддерживает:

* **функциональное программирование**
* **декораторы**
* **замыкания**
* **функции высшего порядка**
* **гибкую организацию кода**

В Python функции можно:

* присваивать переменным
* хранить в структурах данных
* передавать как аргументы
* возвращать из других функций
* создавать во время выполнения программы
